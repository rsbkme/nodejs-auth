## ⚡ Quick Start

```bash
git clone https://github.com/rsbkme/nodejs-auth.git
```

then

```bash
npm install
```

```bash
npm start
```

## 🌲 Environment Varibles

- `PORT` ➡ Express Server PORT
- `DB_URI` ➡ MongoDB URI
- `SECRET` ➡ Your Auth Token Secret
